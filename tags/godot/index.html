    3D Tile System in Godot

    This is a short article to bring some insight into how we implemented a simple 3D tile system in Godot. The code is in C#. Enjoy! Overview In essence, the entire map is just a 3D array where each cell maps to an abstract Tile class: Tile[,,] Tiles; // [column, row, floor] The abstract Tile class defines the common data and behaviour that all tiles in the game should have. Arguably the most important piece of data that each tile has is its position in the 3D array, and its Load method which simply loads a 3D model into the game based on its cell position....
